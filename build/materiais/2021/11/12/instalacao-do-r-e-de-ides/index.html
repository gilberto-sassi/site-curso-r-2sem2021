Instalação do R e de IDE's

Na próxima aula, vamos usar o R localmente, e você precisa instalar uma IDE (editor de texto com capacidades adicionais para facilitar a programação em linguagem R) para acompanhar a aula caso você não queira ou não possa realizar instalações, você pode continuar usando o google colaboratory ou o rstudio.cloud.

Neste curso, vamos utilizar o RStudio.

Instalação do R

Para instalar o R em sua máquina, siga as instruções em cran.r-project.org.

Instalação de IDE

Estamos sugerindo que você use o RStudio, mas sinta-se a vontade para usar qualquer IDE de sua escolha e que seja adequada para a linguagem R não use o bloco de notas.

RStudio

Depois de ter instalado o R, siga as instruções de instalação em www.rstudio.com.
